Stages

    Dans cette partie vous retrouverez les stages que j'ai effectué depuis le début de ma formation.
    En ce jour, j'ai effectué 2 stages en entreprise.
    Le premier dans une entreprise qui s'appelle GTD Internationnal et le second dans une entreprise qui s'appelle Emagma.

GTD International

Année : 2022

Amélioration de l'intranet de l'entreprise

Informations

Dans ce stage, je me suis occupé de l'intranet de l'entreprise en y apportant des améliorations en fonction des demandes des personnes y travaillant.
J'ai dû aussi utiliser Gitlab pour permettre à mon maitre de stage de vérifier le code que je modifiais ou encore pour créer les tickets qui ressence toute les modifications à faire.

Emagma

Année : 2023

Création d'un BOT Discord et API

Informations

Dans ce stage, j'ai participé a la création d'un BOT Discord qui sert à l'entreprise de stocker des rappels personnalisés et qui les renvoies en message privé.
Le Bot Discord affiche aussi dans le salon général de l'entreprise les personnes qui doivent répondre au téléphone.
Pour stocker toutes les données j'ai créé une API sur ApiPlatforme qui relie le bot discord à la base de donner. La base de donné est une base de données MySQL créé sur PhpStorm.
